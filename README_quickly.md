# 光鸭快捷操作

跨目录记录光鸭云盘页面中勾选的文件/文件夹，并在浮窗中按目录分组展示，支持批量移动、删除、解压等操作。

## 功能

- **跨目录勾选记录**：在光鸭云盘中勾选文件/文件夹后，切换到其他目录时勾选记录会被保留。
- **浮窗分组展示**：右侧浮窗按目录分组显示已勾选的文件，支持展开/收起。
- **实时同步**：自动监听页面勾选变化，实时同步到浮窗列表。
- **扫描勾选**：针对虚拟滚动列表，提供"扫描勾选"功能自动滚动并捕获所有已勾选项目。
- **批量移动**：将选中的文件批量移动到其他目录。
- **批量删除**：将选中的文件批量删除。
- **批量解压**：将选中的压缩文件（zip/rar/7z/tar/gz 等）批量解压到指定目录。
- **持久化存储**：勾选记录跨页面刷新持久保存，关闭浏览器后重新打开仍可恢复。
- **浮窗拖拽**：浮窗支持拖拽移动，位置会自动保存。
- **暂停/恢复监听**：可以随时暂停或恢复自动同步。

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 或类似油猴扩展
2. 将此 `.user.js` 文件添加到油猴中启用

## 使用方法

### 基本操作

1. 在光鸭云盘（`guangyapan.com`）页面中勾选需要操作的文件/文件夹
2. 浮窗会自动显示已勾选的项目，按目录分组
3. 切换目录后，之前的勾选记录会保留在浮窗中
4. 点击浮窗中的 `✕` 按钮可移除单个文件或清空整个目录组

### 批量操作

- **扫描勾选**：点击"扫描勾选"按钮，脚本会自动滚动页面并捕获所有已勾选的项目（适用于虚拟滚动列表）。
- **批量移动**：点击"批量移动"按钮，选择目标目录后确认，文件将被移动到指定目录。
- **批量删除**：点击"批量删除"按钮，确认后文件将被删除。
- **批量解压**：点击"批量解压"按钮，选择目标目录后确认，选中的压缩文件将被解压到指定目录（支持串行解压，带进度显示）。

### 浮窗控制

| 按钮 | 说明 |
|------|------|
| `⏸` / `▶` | 暂停/恢复自动监听勾选变化 |
| `−` | 收起浮窗（再次点击 FAB 按钮展开） |
| 浮窗标题栏 | 拖拽可移动浮窗位置 |

## 油猴菜单命令

| 命令 | 说明 |
|------|------|
| `光鸭快捷操作：立即同步当前目录` | 手动触发当前目录勾选同步 |
| `光鸭快捷操作：扫描当前目录全部勾选` | 自动滚动扫描并捕获所有已勾选项目 |

## 支持的压缩格式

zip, rar, 7z, tar, gz, bz2, xz, tgz, zst, iso, cab, arj, lzh, lz, zstd

## 技术说明

### Token 管理

- 与"光鸭云盘 - 磁力链接云添加"脚本共享 `__GUANGYA_CLOUD_QUICKLY_AUTH__` 存储键
- 任一脚本在光鸭页面嗅探到 `Authorization` 即可被两个脚本复用
- 也支持通过油猴菜单手动设置 Token

### 数据持久化

- 勾选记录存储于 `__GYP_CROSS_SEL_RECORDS_V1__` 键
- 浮窗位置存储于 `__GYP_CROSS_SEL_POSITION__` 键
- 暂停状态存储于 `__GYP_CROSS_SEL_POSITION___PAUSED` 键

### API 接口

| 接口 | 路径 | 说明 |
|------|------|------|
| 文件列表 | `POST /userres/v1/file/get_file_list` | 获取目录文件列表 |
| 移动文件 | `POST /userres/v1/file/move_file` | 批量移动文件 |
| 删除文件 | `POST /userres/v1/file/delete_file` | 批量删除文件 |
| 解压文件 | `POST /userres/v1/decompress_files` | 创建解压任务 |
| 任务状态 | `POST /userres/v1/get_task_status` | 查询任务状态 |
| 解压状态 | `POST /userres/v1/query_decompress_status` | 查询解压进度 |

## 注意事项

- 脚本仅在 `guangyapan.com` 域名下生效
- 批量操作前会进行二次确认，防止误操作
- 批量移动/删除/解压任务为异步执行，会轮询任务状态并显示进度
- 扫描勾选功能适用于虚拟滚动列表，可捕获不可见区域的勾选项目
- 如需调试，可在浏览器控制台访问 `window.__gypCrossSelDebug` 对象查看内部状态
